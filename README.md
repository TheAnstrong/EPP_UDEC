# INSTRUCCIONES DE LANZAMIENTO - SISTEMA EPP

## Instalación
Abrir una terminal en la carpeta del proyecto y ejecutar:
pip install -r requirements.txt

## Primera vez
Ejecutar para crear el acceso de administrador:
python init_db.py

## Ejecución
Para abrir el programa, ejecutar siempre:
python main_gui.py
